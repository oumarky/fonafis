Yes, it works with any theme that is compatible with Elementor.

= Can I control the scrolling speed and direction? =  
Yes, you can adjust speed, direction (left, right, up, down), and spacing (gap) for every marquee widget.

= How many items should I add for best performance? =  
For a smooth and seamless experience, we recommend **8â€“10 items per marquee**. Adding too few items may break the continuous scrolling effect.

= Can I add links or lightbox to images and text? =  
Yes, the Image Marquee and Custom Text items support **custom links, media file lightbox, or no link** as per your preference.

= Does the plugin support lazy loading? =  
Yes, you can enable lazy loading to improve page speed and performance.

= Can I pause the marquee on hover? =  
Absolutely! Each widget has a **pause-on-hover option** to allow users to read content easily.
