== Installation ==

Please remember you MUST have [Elementor Page Builder Plugin](https://wordpress.org/plugins/elementor/) installed to be able to use this plugin.

1. Upload the <strong>elfsight-addons-for-elementor.zip</strong> file via the plugin page of WordPress by clicking ‘Add New’ and selecting the zip from your local computer.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. You can find Elfsight Addons under the category “General” on your Elementor element/widget list.
